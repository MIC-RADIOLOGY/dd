# Zimbabwean Direct Deposit Analysis

A Streamlit application for analyzing direct deposit payments in Zimbabwe.  
Supports both **medical aid** and **individual payments** and provides detailed reporting, charts, and Excel export.

## Features

- **Upload Direct Deposit Data** in Excel or CSV format (columns: Date, Payer, Amount required)  
- **Dynamic Zimbabwean Medical Aid List** (optional upload)  
- **Interactive Filters**:
  - Select specific medical aids
  - Filter by date range
- **Top 3 Medical Aids** and **Top 3 Individual Payers**
- **Top 10 Charts** for medical aids and individual payments
- **Month-over-Month Comparison**
- **Payment Trends** and **Monthly Growth Percentages**
- **Excel Report Download** with all analyses and charts included
